# 各版本历史

## 梦幻群侠传5


#### BUG版、四启动版
作者：子犯  
版本：2016年
主角：篮子  
门派：经典12门派  
角色：经典15角色  
玩法：剧情、钟馗捉鬼、门派闯关、副本（乌鸡国、水陆大会、车迟国、四门绝阵）  
说明：四启动版比BUG版多了两个适配WindowsXP系统的启动器

#### 梦幻群侠传5
作者：群主  
版本：2016年——至今  
说明：群主从作者接手的源码。在保留一切玩法和存档**原滋原味**的基础上， 对游戏进行优化和修复。  
其他：为了实现**分辨率自由**，更换了游戏引擎（包括后续的资料片、DLC、唯美版、白泽版、日期版），但是部分电脑会莫名白屏或无法运行，由于技术有限，故无解。

## 日期版

#### 梦幻群侠传5.17.2.25 (及以前)（17年2月25日，即发布日期，故名日期版）
作者：群主  
版本：2017年  
**配角**：文堂（跑腿的意思）  
说明：基于梦幻群侠传5，沿用梦幻群侠传5的剧情，在此基础上添加**比武**和**帮战**  
内置：内置黑科技，直接刷经验、刷金币、刷装备  
标记：画面有"**游戏免费勿商用**"标记，帮派是**射狼传说**标记  
**嫑坑**：宝宝资质面板有“免费”、“嫑坑”标记（免费游戏，不要被坑）

#### 日期版

## 小草终结版

#### 浪西游
作者：群主  
版本：2018年  
主角：剑侠客  
说明：基于日期版。添加擂台挑战、蜈蚣精挑战  
终结：往后都叫**小草终结版**了  
安卓：真正的**浪西游**，群主用来写安卓版了

#### 浪西游之小草终结版（儿童节、佐助、雏田图标）
作者：小草  
版本：2018年——2019年  
主角：忘了  
说明：草儿拿浪西游改着玩（被草儿玩坏了，故名小草终结版）

#### 小草终结版
作者：群主  
版本：2018——2022年(220531)  
主角：剑侠客  
门派：15门派  
角色：18角色  
玩法：剧情挑战使者、钟馗捉鬼、门派闯关、封妖、比武、帮战、地煞星  
剧情：仿官方剧情（截图官方剧情对话一句一句打字真的累人），已至生死劫（大概设想总进度的80%）  
挂机：**自动**钟馗捉鬼


#### 黑子的足球
作者：群主  
版本：2023年(**3825**即23年8月25日)  
说明：电脑中了一次勒索病毒，而仓库留下的源码的**剧情进度几乎全部丢失**，故留此版本作为小草终结版的**纪念版**


#### 终结版
作者：群主  
版本：2023——至今(目前版本**3C22**即23年12月22日)  
重写：**完全脱离**梦幻群侠传5，全部重写，包括游戏引擎和游戏数据  
剧情：剧情进度丢失后，**重写剧情**好累人  
玩法：钟馗捉鬼、封妖、门派闯关、**野外PK**、**地煞星**  
挂机：自动师门、**自动打图挖图**
